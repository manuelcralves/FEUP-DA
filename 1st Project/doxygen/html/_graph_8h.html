Deletes a dynamically allocated 2D double matrix.

This function is responsible for deleting a dynamically allocated 2D double matrix. It takes a pointer to the matrix 'm' and the number of rows/columns 'n' as input. The function iterates over each row and deletes the dynamically allocated memory for each row, and then deletes the memory allocated for the matrix itself.
